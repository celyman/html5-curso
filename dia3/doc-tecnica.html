Función calcularAreaCuadrado

¿Qué hace esta función?

calcularAreaCuadrado Esta función recibe un número que representa el lado de un cuadrado y devuelve su área

La idea es muy simple

área = lado * lado o también puedes escribir como A=L2

Donde

A
Área del cuadrado
L
Longitud del lado
L2
El lado multiplicado por sí mismo

Código de la función


            function calcularAreaCuadrado(lado) { 
              let area = lado * lado; 
              return area; 
            }

Explicación línea por línea

1. Declaración de la función

function calcularAreaCuadrado(lado)

Aquí estamos creando una función llamada calcularAreaCuadrado

  • function es la palabra que JavaScript usa para crear funciones
  • calcularAreaCuadrado es el nombre de la función
  • lado es el dato de entrada

2.Variable matemática sencilla

let area = lado * lado;

Aquí se crea una variable llamada area

  • let sirve para guardar un valor
  • lado * lado significa multiplicar un lado por sí mismo

3. Resultado de salida

  • return area;
  • return

    significa que la función entrega el resultado final

Como invocarla desde la terminal

Para ejecutar esta función, puedes usar Node.js.

  • 1. Crea un archivo

    Guarda este código en un archivo llamado area.js 

    • 
                function calcularAreaCuadrado(lado) { 
                  let area = lado * lado; 
                  return area; 
                } 
                
                let resultado = calcularAreaCuadrado(5);
                console.log("El área del cuadrado es:", resultado);

  • 2. Abre la terminal

    • Ubícate en la carpeta donde guardaste el archivo

  • 3. Abre la terminal presionando las teclas Windows + ñ y luego ejecutas el archivo 

    •  node area.js

  • 4. Salida en terminal

    • El área del cuadrado es 25
